Grid Lab

This lab demonstrates various configurations of a reusable data grid component using Angular Material. Learn how to customize tables with different features like search, pagination, and actions.

Basic Table Minimal configuration without extra features
This example shows a simple table configuration with:
  • Basic title
  • No search functionality
  • No pagination
  • No action buttons
Advanced Table Full-featured configuration with all options enabled
This example demonstrates all available features:
  • Search functionality
  • Pagination controls
  • Action buttons
  • Status tabs
Paginated Table Table with pagination focus
This example shows pagination features:
  • Pagination controls
  • Customizable page size
  • No search or tabs
  • With actions
Tabs-Only Table Focus on status management
Demonstrates tab-based status management:
  • Status tabs navigation
  • Custom tab labels
  • No search or pagination
